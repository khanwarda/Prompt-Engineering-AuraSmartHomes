# AURA Smart Homes - AI-Generated Real Estate Advertisement

## 📌 Project Overview
This project demonstrates the use of advanced Prompt Engineering to generate high-quality, marketing-ready visual assets and copy for a fictional premium real estate brand, **AURA Smart Homes**. 

As an AI Automation Engineer, this project highlights my ability to leverage Generative AI tools to bridge the gap between technical logic and creative business solutions.

## 🛠️ Tech Stack & Tools
* **AI Image Generation:** Gemini AI (Image Generation Model)
* **Prompt Optimization:** Custom Prompt Engineering Formulas & AIPRM
* **Content Generation:** Gemini AI (LLM for Ad Copy)
* **Design Strategy:** Visual Hierarchy & Professional Typography Integration

## 🧠 Prompt Engineering Formula
The advertisement was generated using a highly structured prompt architecture:
* **Role:** Expert AI Graphic Designer
* **Task:** Design a futuristic luxury smart-villa advertisement.
* **Style:** Hyper-realistic, architectural photography, sleek & futuristic.
* **Platform:** Facebook Feed (1:1 Square).

## 📝 The Master Prompt
> "Create a high-resolution Facebook Feed advertisement image (1:1 Square). The scene features a hyper-realistic, high-end architectural photography shot of a futuristic luxury smart-villa at twilight. The house features sleek geometric lines, massive floor-to-ceiling glass windows, and warm golden interior lighting that contrasts beautifully with the cool blue evening sky. A subtle, glowing holographic UI interface hovers near the sleek front door. Add professional typography: Top Center: 'AURA SMART HOMES', Main Headline: 'THE FUTURE OF LIVING IS HERE.', Sub-headline: 'Experience Aura Smart Homes. 100% AI-Automated Luxury Villas.', Bottom Tagline: 'Where Intelligence Meets Elegance.', Contact: '+92 300 1234567 | www.aurahomes.com | Book Your Virtual Tour Today'"
